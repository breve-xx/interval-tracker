# TASK-0015: Brand Glyph in Header

## Goal

Add a single `activity` icon (Lucide) immediately before the "Interval Tracker" brand
text in the sticky header, reinforcing the application's identity with a minimal,
purposeful glyph.

---

## Context & Design Constraints

- **Icon**: `activity` from Lucide — a pulse/waveform line that directly evokes time
  series monitoring. Confirmed present in Lucide 1.14.0 (the version resolved by the
  CDN `@latest` tag).
- **Placement**: inside `.app-header__brand`, to the left of the text, in a flex row.
- **Sizing**: `1.1em` so the glyph sits slightly larger than the surrounding label text
  (`0.9rem`) without dominating it.
- **Colour**: inherits `currentColor` from `.app-header__brand` (`var(--text-muted)`),
  so it theme-switches for free.
- **No new JavaScript**: `refreshIcons()` is already called at the end of `initUI()`,
  which materialises all `<i data-lucide="…">` placeholders present at load time.

---

## Implementation Steps

### Step 1 — `src/index.html`

Replace the plain `<span>` brand with a flex row containing the icon and the text:

```html
<span class="app-header__brand">
  <i data-lucide="activity"></i>
  Interval Tracker
</span>
```

### Step 2 — `src/css/styles.css`

Update `.app-header__brand` to lay out icon + text as an inline flex row:

```css
.app-header__brand {
  display     : inline-flex;
  align-items : center;
  gap         : 0.4em;
  font-size   : 0.9rem;
  font-weight : 600;
  color       : var(--text-muted);
  letter-spacing: 0.02em;
  text-transform: uppercase;
}
.app-header__brand svg.lucide {
  width  : 1.1em;
  height : 1.1em;
}
```

### Step 3 — No new tests, no coverage impact

No business logic is introduced. All existing tests continue to pass.
